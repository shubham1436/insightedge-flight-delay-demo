InsightEdge Python API Demo

Spark streaming application does flight delay prediction and displays predictions from Kafka stream.

First part of the application is flight prediction based on user input. Enter needed values into the inputs and click 'Predict'. Input data will be sent to the Kafka topic and processed by trained model. Later you will see a prediction in 'Submitted flights' table.

Second part is showing flights prediction in the 'Streamed flights'. Data is sent to the Kafka topic previously at the 8th step of README file.
